SAM Crack Segmentation Experiments: Prompt Point Strategies and Performance Evaluation

Mar 14, 2025
简体中文

Experiment Objectives and Background

This experiment aims to explore the application of the Segment Anything Model (SAM) in crack image segmentation tasks, with a focus on investigating how different prompt point strategies affect segmentation quality. Through systematic testing of positive and negative points, we have summarized a series of practical optimization strategies.

Prompt Point Strategy Experiments

The Optimization Role of Negative Points

In practical testing, when manually inputting relatively accurate crack prompt points, and when the crack clearly divides the image into two parts, using only positive points may cause SAM to incorrectly identify large road surface areas as segmentation targets. In such cases, adding two negative points in the mask region can significantly improve segmentation results and accurately identify the crack area.

Key Finding: Properly setting a small number of negative points can effectively constrain SAM’s segmentation range and significantly improve crack identification accuracy.

Impact of Positive Point Distribution

When positive points fail to adequately cover the crack region, incomplete edge recognition occurs, leading to a large number of incorrect segmentations:

Optimization Recommendation: The distribution of positive points should closely match the actual shape of the crack, ensuring complete coverage of the crack path.

def get_local_direction(x, y, window_size=5):
    try:
        # Extract local window
        y1 = max(0, y - window_size)
        y2 = min(skeleton.shape[0], y + window_size + 1)
        x1 = max(0, x - window_size)
        x2 = min(skeleton.shape[1], x + window_size + 1)

        # Check window size
        if y2 <= y1 or x2 <= x1:
            return None

        local_skeleton = skeleton[y1:y2, x1:x2]

        # Check if local window is empty
        if np.sum(local_skeleton) == 0:
            return None

        # Calculate gradient using Sobel operator
        gy = ndimage.sobel(local_skeleton, axis=0)
        gx = ndimage.sobel(local_skeleton, axis=1)

        # Ensure relative coordinates are within valid range
        rel_y = min(y - y1, local_skeleton.shape[0] - 1)
        rel_x = min(x - x1, local_skeleton.shape[1] - 1)

        # Calculate local direction angle (radians)
        if np.abs(gx[rel_y, rel_x]) < 1e-10 and np.abs(gy[rel_y, rel_x]) < 1e-10:
            return None

        angle = np.arctan2(gy[rel_y, rel_x], gx[rel_y, rel_x])
        return angle

Positive-Negative Point Ratio Experiments (Random Sampling)

Through systematic experiments, the following conclusions were drawn:

  1. Impact of Prompt Point Quantity: Too many prompt points introduce noise and reduce segmentation accuracy.
  2. Value of Negative Points: An appropriate amount of negative points can effectively clarify segmentation targets, performing better than using only positive points.

  1. Mask Region Limitations: Even with negative points set, certain regions’ masks may still be retained and cannot be completely removed.

  1. Sampling Strategy Optimization: Random sampling may cause sampling points to concentrate on one side; adjusting the brightness threshold (currently 10%) can improve distribution uniformity.

Summary and Recommendations

Key Findings

  1. Negative Point Strategy: Reasonable use of negative points can effectively constrain SAM’s segmentation range and significantly improve crack identification accuracy.
  2. Positive Point Distribution: Positive points should fully cover the crack region and match the actual shape of the crack.
  3. Sampling Optimization: Skeleton-based directional sampling strategy outperforms random sampling and can generate more targeted negative points.
  4. Point Quantity Balance: The number of prompt points should be moderate; too many will introduce noise.

Practical Application Recommendations

  • In crack segmentation tasks, it is recommended to adopt a combination strategy of “small number of positive points + appropriate number of negative points”
  • Prioritize using skeleton-based directional negative point sampling methods
  • For complex crack morphologies, adopt a multi-point gradual refinement interaction approach
  • Pay attention to performance differences between local models and official demos, and perform post-processing optimization when necessary

Future Research Directions

  • Explore intelligent strategies for adaptive negative point quantity and position
  • Study optimal prompt point configurations for different crack types (horizontal/vertical/mesh)
  • Combine traditional image processing methods with SAM for hybrid segmentation strategies
