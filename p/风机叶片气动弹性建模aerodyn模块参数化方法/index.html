风机叶片气动弹性建模：AeroDyn模块参数化方法

Feb 26, 2025
English

系列导读：本系列文章将系统介绍 OpenFAST 风机仿真软件的使用方法，涵盖 AeroDyn 空气动力学模块、ElastoDyn 结构动力学模块、控制系统集成等内容。本文为系列开篇，详细介绍 AeroDyn 空气动力学模块的配置与使用方法。

先导知识：致动线理论（Actuator Line Theory）是风力机空气动力学建模的重要方法，将叶片简化为沿展向分布的线源，通过计算叶片元素上的气动力来模拟叶片与流场的相互作用。该方法在保持计算效率的同时，能够较好地捕捉叶片的气动特性。

参考文献

  1. Jonkman, J.M., Hayman, G.J., Jonkman, B.J., Damiani, R.R., Murray, R.E., n.d. AeroDyn v15 User’s Guide and Theory Manual. Renewable Energy.
  2. OpenFAST官方文档：https://openfast.readthedocs.io/en/main/index.html

AeroDyn模块详解

模块简介

  1. AeroDyn 是一个时域空气动力学计算模块，既可作为独立程序运行以计算未与 OpenFAST 耦合的风机气动响应，也可集成到 OpenFAST 框架中实现多物理场耦合仿真；

  2. 该模块主要用于计算叶片和塔筒的空气动力载荷，其核心理论基础为致动线理论（Actuator Line Theory）；

  3. AeroDyn 由六个子模型构成，分别负责：

    • 转子尾流与诱导速度计算（rotor wake/induction）
    • 叶片翼型空气动力学（blade airfoil aerodynamics）
    • 塔筒对叶片局部流场的干扰效应（tower influence on the fluid local to the blade nodes）
    • 塔筒与机舱的气动阻力（tower and nacelle drag）
    • 气动噪声计算（aeroacoustics）
    • 叶片、轮毂、机舱和塔筒的浮力效应（buoyancy）

输入文件配置

  1. 基本单位制：千克-米-秒-牛顿（kg, m, s, N）

通用选项设置

  1. DTAero: 空气动力学计算时间步长，建议至少设置200个步长以保证计算稳定性；

  2. Wake_mod（尾流模型选择）:

    • 设置为 0：不考虑诱导速度效应；
    • 设置为 1：基于叶素-动量理论（BEM）计算诱导速度；
    • 设置为 3：采用自由涡尾流模型（free vortex wake model），注意在线性分析阶段不能设置为 3；

  3. Twrpotent（塔筒势流效应）:

    • 设置为 0：不考虑塔筒对叶片局部流场的潜在影响；
    • 设置为 1：采用标准势流模型（standard potential-flow model）；
    • 设置为 2：在标准势流模型基础上增加 Bak 修正；

建模参数推荐设置

  1. NumVBlNds（叶片节点数）:

    • 建议设置在 10-20 之间，以平衡计算精度与计算成本；

  2. 运行状态风机叶片的仿真配置建议：

    • 推荐启用动态 BEM 模型（Wake_Mod = 1）和非定常空气动力学模型（AFAeroMod = 2）；
    • 启用 SkewMod = 2、TipLoss = TRUE、HubLoss = TRUE、TanInd = TRUE；
    • 注意：当偏航误差大于 45 度时，SkewMod = 2 设置无效；
    • BEM 非线性求解与入流角相关，IndToler 建议保持默认值 DEFAULT；

输出设置配置

  1. AeroDyn 支持沿塔筒 9 个节点和沿叶片 9 个节点的气动力学和运动学量输出；

  2. BldNd_BladesOut: 指定需要输出结果的叶片数量；

  3. BldNd_BlOutNd: 指定需要输出的节点编号；

  4. 叶片节点气动力输出选项：

    • “to plane”（相对于转子平面）：

      • Fx：垂直于叶片旋转平面的力，适用于评估整体叶片性能和计算全局载荷；
      • Fy：平行于叶片旋转平面的力；

    • “to chord”（相对于叶片弦线）：

      • Fn：垂直于叶片局部弦线的力；
      • Ft：平行于叶片局部弦线、沿前缘到后缘方向的力，影响叶片扭转变形；
      • 适用于分析局部叶片结构响应和变形；

import os
import subprocess
import pandas as pd
import time
from openfast_toolbox.io import FASTInputFile, FASTOutputFile


def split_nodes_into_groups(nodes, group_size=9):
    """将节点列表分成每组最多9个节点"""
    return [nodes[i:i + group_size] for i in range(0, len(nodes), group_size)]


def modify_aerodyn(aero_file_path, node_group, output_path):
    """修改AeroDyn输入文件"""
    f = FASTInputFile(aero_file_path)

    # 设置输出节点数量
    num_nodes = len(node_group)
    f['NBlOuts'] = num_nodes  # 设置输出节点数量

    # 设置节点
    f['BlOutNd'] = ','.join(map(str, node_group))

    # 生成输出列表
    output_list = [""]  # 添加初始换行
    for blade in [1, 2, 3]:
        for node_index, node_value in enumerate(node_group, 1):  # 从1开始计数
            output_list.extend([
                f"B{blade}N{node_index}Fn",
                f"B{blade}N{node_index}Ft",
                f"B{blade}N{node_index}Mm"
            ])

    f['OutList'] = output_list
    f.write(output_path)
    return output_path


def run_openfast(fast_exe_path, fst_file_path):
    """运行OpenFAST"""
    try:
        working_dir = os.path.dirname(fst_file_path)
        os.chdir(working_dir)
        command = [fast_exe_path, os.path.basename(fst_file_path)]
        subprocess.run(command, check=True)
        print("OpenFAST simulation completed successfully")
        return True
    except subprocess.CalledProcessError as e:
        print(f"Error running OpenFAST: {e}")
        return False


def extract_blade_data(outb_file, node_group):
    """从.outb文件提取数据"""
    df = FASTOutputFile(outb_file).toDataFrame()
    blade_data = {1: {}, 2: {}, 3: {}}
    time = df['Time_[s]']

    for blade in [1, 2, 3]:
        data_dict = {'Time_[s]': time}
        # 枚举node_group获取索引和值
        for node_index, node_value in enumerate(node_group, 1):  # 从1开始计数
            for measure in ['Fn', 'Ft', 'Mm']:
                if measure == 'Mm':
                    col_name = f'B{blade}N{node_index}{measure}_[N-m/m]'
                else:  # measure is 'Fn' or 'Ft'
                    col_name = f'B{blade}N{node_index}{measure}_[N/m]'
                if col_name in df.columns:
                    data_dict[f'Node{node_value}_{measure}'] = df[col_name]
                    print(f"Found data for {col_name}")
                else:
                    print(f"Column {col_name} not found")
        blade_data[blade] = pd.DataFrame(data_dict)

    return blade_data


def merge_blade_data(all_runs_data):
    """合并多次运行的数据"""
    merged_data = {1: {}, 2: {}, 3: {}}

    for blade in [1, 2, 3]:
        # 使用第一次运行的时间数据
        merged_df = pd.DataFrame({'Time_[s]': all_runs_data[0][blade]['Time_[s]']})

        # 合并所有运行的数据
        for run_data in all_runs_data:
            cols_to_merge = [col for col in run_data[blade].columns if col != 'Time_[s]']
            merged_df[cols_to_merge] = run_data[blade][cols_to_merge]

        merged_data[blade] = merged_df

    return merged_data


def save_to_excel(blade_data, output_path):
    """保存数据到Excel文件"""
    with pd.ExcelWriter(output_path) as writer:
        for blade_num, data in blade_data.items():
            data.to_excel(writer, sheet_name=f'Blade_{blade_num}', index=False)
    print(f"Data saved to {output_path}")


def main():
    # 设置路径
    work_dir = 'E:/Study/openfast/r-test-3.5.3/glue-codes/openfast/5MW_Test_Weihao'
    fast_exe_path = 'E:/Study/openfast/r-test-3.5.3/glue-codes/openfast/5MW_Test_Weihao/openfast_x64.exe'
    fst_file = os.path.join(work_dir, '5MW_Land_BD_DLL_WTurb.fst')
    aero_file = os.path.join(work_dir, 'NRELOffshrBsline5MW_Onshore_AeroDyn15.dat')
    output_excel = os.path.join(work_dir, 'blade_data_all_nodes.xlsx')

    # 定义所有需要的节点
    all_nodes = list(range(1, 20))
    node_groups = split_nodes_into_groups(all_nodes)

    all_runs_data = []

    for i, node_group in enumerate(node_groups):
        print(f"\nProcessing node group {i + 1}: {node_group}")

        # 修改后的气动文件名
        modified_aero_file = os.path.join(work_dir, f'NRELOffshrBsline5MW_Onshore_AeroDyn15_group_{i + 1}.dat')

        # 修改气动文件
        modify_aerodyn(aero_file, node_group, modified_aero_file)

        # 修改FST文件中的AeroDyn文件路径
        fst = FASTInputFile(fst_file)
        fst['AeroFile'] = f'"{os.path.basename(modified_aero_file)}"'
        modified_fst_file = os.path.join(work_dir, f'5MW_Land_DLL_WTurb_group_{i + 1}.fst')
        fst.write(modified_fst_file)

        # 运行OpenFAST
        if run_openfast(fast_exe_path, modified_fst_file):
            # 提取数据
            outb_file = os.path.join(work_dir, f'5MW_Land_DLL_WTurb_group_{i + 1}.outb')
            run_data = extract_blade_data(outb_file, node_group)
            all_runs_data.append(run_data)

            # 等待一小段时间确保文件写入完成
            time.sleep(1)

    # 合并所有运行的数据
    if all_runs_data:
        merged_data = merge_blade_data(all_runs_data)
        # 保存最终数据
        save_to_excel(merged_data, output_excel)
    else:
        print("No data was collected from any runs")


if __name__ == "__main__":
    main()
  • 避坑点1：OutList需要添加初始换行，否则第一个变量将不会输出；
  • 避坑点2：BlOutNd需要使用逗号来连接变量，直接赋值会导致出现括号，造成报错；
