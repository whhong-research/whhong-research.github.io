实验图像

添加负点来优化分割结果

  • 当人工输入较为完美的裂缝提示点，且裂缝将图像分为明显两部分时，只输入正点可能导致SAM错误将大面积的一部分路面作为识别结果；此时，如果添加两个负点在掩码区域，将会出现较好识别裂缝的情况：

  • 猜想结论：正确少量的负点能很大程度优化sam的分割结果；

正点的数量和分布影响

  • 当正点未能较好地分布在裂缝上时，会出现裂缝边缘识别不全的情况，产生大量错误识别图案的情况：

  • 猜想结论：正点的覆盖区域尽量贴近裂缝形状

      def get_local_direction(x, y, window_size=5):
    try:
        # 提取局部窗口
        y1 = max(0, y - window_size)
        y2 = min(skeleton.shape[0], y + window_size + 1)
        x1 = max(0, x - window_size)
        x2 = min(skeleton.shape[1], x + window_size + 1)

        # 检查窗口大小
        if y2 <= y1 or x2 <= x1:
            return None

        local_skeleton = skeleton[y1:y2, x1:x2]

        # 检查局部窗口是否为空
        if np.sum(local_skeleton) == 0:
            return None

        # 使用Sobel算子计算梯度
        gy = ndimage.sobel(local_skeleton, axis=0)
        gx = ndimage.sobel(local_skeleton, axis=1)

        # 确保相对坐标在有效范围内
        rel_y = min(y - y1, local_skeleton.shape[0] - 1)
        rel_x = min(x - x1, local_skeleton.shape[1] - 1)

        # 计算局部方向角度（弧度）
        if np.abs(gx[rel_y, rel_x]) < 1e-10 and np.abs(gy[rel_y, rel_x]) < 1e-10:
            return None

        angle = np.arctan2(gy[rel_y, rel_x], gx[rel_y, rel_x])
        return angle

    • 从垂直于angle的方向进行采样，并删去部分特别接近裂缝骨架的点: distance为设置的距离（40像素），min_skeleton_distance设置为10像素；

正点和负点的比例影响（随机采样）

  • 猜想1：提示点的增多会带来噪声；
  • 猜想2：少量负点能够明确需要分割的对象，效果优于0负点；

  • 猜想3：即使有些区域设置了负点，得出的掩码仍然不会剔除该区域；

  • 弊端1：随机采样可能会出现采样点大量落在单独一侧的情况，调整亮度阈值（当前为10%）可能会有所改善；
