Feb 24, 2025
English

系列导读：本系列文章将系统介绍计算机视觉在结构健康监测中的应用，涵盖图像预处理、特征提取、深度学习模型构建等核心技术。作为系列开篇，本文重点阐述裂缝检测中的骨架化技术及其在精细化分割中的应用。

1. 任务背景与动机

1.1 研究背景

在结构健康监测领域，裂缝检测是评估混凝土结构完整性的关键环节。传统人工巡检方式存在效率低下、一致性难以保证等局限性。近年来，基于计算机视觉的自动化裂缝检测方法因其高效、客观的优势，受到学术界和工业界的广泛关注。

1.2 技术路线

本项目采用的技术路线如下：

  1. 图像生成：利用 Diffusion 模型生成无裂缝参考图像
  2. 差异分析：通过 SSIM（结构相似性指数）计算重建图像与原始图像的差异图
  3. 粗糙分割：将差异图作为初始的裂缝分割结果
  4. 精细化分割：将分割结果作为提示输入 SAM（Segment Anything Model）进行精细化分割

2.1 阈值分割的局限性

2.2 骨架化的优势

SAM 模型接受两种提示方式：Mask 提示和 Point 提示。Mask 提示可视为密集的 Point 提示集合。

核心思路：通过骨架化提取裂缝的中心线特征点，作为 SAM 模型的 Prompt 进行精细化分割。

该方法具有以下优势：

  • 有效抑制噪声干扰
  • 精确提取裂缝中心线
  • 生成更具语义信息的提示点

3. 实现代码详解

3.1 代码功能概述

  1. 图像预处理：将图像 resize 为 (1024, 512) 尺寸，便于统一定义形态学去噪的 kernel 大小和连通域过滤阈值

def adaptive_threshold1(self, image):
    """
    自适应阈值分割：选取图像中最暗的5%像素作为裂缝区域
    
    参数:
        image: 输入图像，可为灰度图或彩色图
    
    返回:
        binary_img: 二值化图像，裂缝区域为白色(255)，背景为黑色(0)
    """
    # 调整图像大小为标准尺寸，便于统一处理
    image = cv2.resize(image, (1024, 512))

    # 转换为灰度图（若输入为彩色图）
    if len(image.shape) > 2:
        gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    else:
        gray = image.copy()

    # 双边滤波：在保持边缘的同时去除噪声
    # d=15: 滤波邻域直径
    # sigmaColor=150: 颜色空间滤波sigma
    # sigmaSpace=150: 坐标空间滤波sigma
    smoothed = cv2.bilateralFilter(gray, d=15, sigmaColor=150, sigmaSpace=150)

    # 将图像展平为一维数组
    flat_img = smoothed.flatten()
    
    # 计算阈值：选取最暗的5%像素的灰度值作为阈值
    # np.percentile计算第5百分位数
    threshold = np.percentile(flat_img, 5)
    
    # 二值化处理：创建全黑背景
    binary_img = np.zeros_like(smoothed)
    # 将暗于阈值的像素设为白色（裂缝区域）
    binary_img[smoothed <= threshold] = 255
    
    return binary_img

def morphological_operations(self, binary):
    """
    形态学操作：去除噪点和小面积连通域
    
    参数:
        binary: 二值化图像
    
    返回:
        closed: 经过形态学操作后的图像
    """
    # 开运算：先腐蚀后膨胀，去除小噪点
    # kernel=(1,1): 使用1x1的核，最小程度的形态学操作
    kernel = np.ones((1,1), np.uint8)
    opening = cv2.morphologyEx(binary, cv2.MORPH_OPEN, kernel)
    
    # 连通域分析：标记并统计每个连通域
    # cv2.connectedComponentsWithStats返回：
    #   num_labels: 连通域数量（包括背景）
    #   labels: 每个像素所属的连通域标签
    #   stats: 每个连通域的统计信息（面积、外接矩形等）
    #   centroids: 每个连通域的质心
    num_labels, labels, stats, centroids = cv2.connectedComponentsWithStats(opening)
    
    # 创建新的图像，只保留大联通域
    morphed = np.zeros_like(binary)
    
    # 遍历所有连通域（从1开始，0是背景）
    for i in range(1, num_labels):
        # 如果连通域面积大于等于阈值，保留
        if stats[i, cv2.CC_STAT_AREA] >= self.min_area:
            # 将该连通域的所有像素设为白色
            morphed[labels == i] = 255
    
    # 闭运算：先膨胀后腐蚀，填充小孔洞
    closed = cv2.morphologyEx(morphed, cv2.MORPH_CLOSE, kernel)
            
    return closed

def skeletonize(self, binary):
    """
    骨架化：提取裂缝的中心线
    
    参数:
        binary: 二值化图像
    
    返回:
        skeleton: 骨架图像，裂缝中心线为白色(255)
    """
    # 确保输入是单通道二值图像
    if len(binary.shape) > 2:
        binary = cv2.cvtColor(binary, cv2.COLOR_BGR2GRAY)      

    # 使用skimage的骨架化算法
    skeleton = skeletonize(binary)

    # 添加后处理步骤（如去除小分支等）
    skeleton = process_skeleton(skeleton)

    # 转换为uint8格式，白色为255
    skeleton = skeleton.astype(np.uint8) * 255
    
    return skeleton

6.1 cv2.distanceTransform 

1

dst = cv2.distanceTransform(src, distanceType, maskSize[, dstType])

参数说明：

  1. src: 输入图像，必须为单通道8位二值图像
  2. distanceType: 距离类型
    • cv2.DIST_L1: 曼哈顿距离
    • cv2.DIST_L2: 欧氏距离
    • cv2.DIST_C: 切比雪夫距离
  3. maskSize: 距离变换掩码大小
    • cv2.DIST_MASK_3: 3x3 掩码
    • cv2.DIST_MASK_5: 5x5 掩码
    • cv2.DIST_MASK_PRECISE: 更精确的算法
  4. dstType: 输出图像类型（可选）
    • cv2.CV_8U: 8位无符号整型
    • cv2.CV_32F: 32位浮点型
    • cv2.CV_64F: 64位浮点型

from skimage.morphology import medial_axis

def medial_axis_skeleton(binary):
    # 返回骨架和距离变换
    skeleton, distance = medial_axis(binary, return_distance=True)
    return skeleton.astype(np.uint8) * 255, distance
